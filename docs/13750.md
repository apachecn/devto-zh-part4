# Kotlin 中的数据类型

> 原文：<https://dev.to/jay_tillu/data-types-in-kotlin-2749>

*   数据类型是指可以在编程语言中使用和操作的数据类型。

*   Kotlin 是一种类似 Java 的静态类型语言。这意味着变量的类型在编译时是已知的。

*   请记住，在 kotlin 中，一切都是对象，我们可以调用任何变量的成员函数和属性。

例如，考虑以下情况:

```
highScore.toInt()
a + b — a.plus(b) 
```

*   这里的`highScore`是一个变量，我们对其调用`toInt()`转换函数。而`a`和`b`也是`a.plus(b)`内部调用的变量。这就是为什么他们称万物为物体。

kotlin 中基本上有三种数据类型:

1.  数字
2.  性格；角色；字母
3.  布尔代数学体系的
