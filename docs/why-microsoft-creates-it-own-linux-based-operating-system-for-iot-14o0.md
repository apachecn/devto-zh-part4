# 为什么微软要为物联网开发基于 Linux 的操作系统。

> 原文：<https://dev.to/xeroxism/why-microsoft-creates-it-own-linux-based-operating-system-for-iot-14o0>

[![iot-car-system-banner](img/7b5c670a395c6bb2afaee774d108b736.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--S2vn52sR--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i1.wp.com/fossnaija.com/wp-content/uploads/2019/06/iot-banner-1.jpg%3Fresize%3D665%252C475%26ssl%3D1)

微软为物联网设备创造了新的操作系统 Azure sphere。你猜怎么着，它基于 [Linux](https://fossnaija.com/linux-at-25-from-grass-to-grace/) 。是的，没错。这是微软首次公开发布运行 Linux 内核的操作系统。

当然，透过表面来看，这并不奇怪。物联网是微软的新地盘。这也是它钟爱的 Windows 操作系统(OS)失败并被 Linux 抛弃的另一个主要领域。目标是为物联网设备提供强大而安全的云连接服务/平台。

微软最新的 OS 是 Windows 10(w10)；它不能用作物联网设备的基础操作系统，因为它对于规格远低于 w10 安装要求的相对较小的设备来说太大了。

### 安全！保安！！保安！！！

将一切都连接到互联网的想法也产生了对这些连接设备的安全性的需求。因为它们可以从多个方面对云基础设施发起网络威胁或攻击。尤其是因为这些设备通常不会被普通用户更新或安装最新的安全补丁。Azure sphere 驱动的设备将随后推出。

[![azure_sphere_model](img/f818826568d7e2fd9d181b5fc440e65d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--5PCVnt07--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i2.wp.com/fossnaija.com/wp-content/uploads/2019/06/azure_sphere_model-1.png%3Fw%3D665%26ssl%3D1)

当然，微软打算通过 azure 云平台向物联网设备提供云服务来赚钱。最近，Azure 已经成为 it 的摇钱树，为其带来了大部分收入。

主要是通过设备制造商的订购，他们决定在他们的产品中使用芯片。该领域的芯片和操作系统(OS)组件将免版税和许可证。

诸如此类的事情不断证明了这样一个事实，即[免费和开源的开发模式](https://fossnaija.com/open-source-powershell-microsoft-loves-linux/)和理念——超越 Linux——对未来有很多承诺。至少按时实现它被证明是其生存的重大举措。如果你不能打败他们，他们会帮助你。【T2![🙂](img/1bb31e891282bfa40812655c9c9ace9e.png)
