# 挽回时间

> 原文:[https://dev.to/megazear7/redeeming-the-time-5db0](https://dev.to/megazear7/redeeming-the-time-5db0)

我最近发现自己在给时间管理提建议。在这次讨论中，“减少浪费的时间，挽回必要的时间”这个短语出现了。我认为这概括了提高生产率的两个主要方法。一些时间同步纯粹是浪费，而另一些时间同步是必要的，但却是无效的。为了提高生产率，需要减少浪费的时间，同时需要补偿必要的时间同步。

然而，什么是浪费的时间，或者就此而言，什么是未得到补偿的时间？首先，它假设了一个目标。如果时间被浪费了，那么就有一个隐含的假设，那就是它没有做什么事情。因此，为了让浪费时间的概念有意义，我们需要有一个计划、过程或我们努力实现的目标。当计划外的和原因不明的干扰打断了我们的努力，我们可以认为时间被浪费了。然而，让我们从理论的迷雾中走出来，用具体的例子来谈论它。

浪费时间最容易的例子是上瘾的手机应用程序；有人在脸书或 YouTube 上无限滚动吗？它确实是无限的，我可以向你保证这一点(尽管我还不能证明这一点...但我肯定会接近)。一般来说，我们不会“计划”在晚饭后的一个小时内，或者在离开健身房之前，或者在我们做这些事情的其他地方，滚动社交媒体或其他形式的应用程序。最常见的是，这些时间是没有计划的，没有说明的，没有产出的，拖着我们偏离轨道，阻止我们达到目标。虽然给出如何克服这个具体例子的详细建议不在我的考虑范围之内，但这一点仍然很清楚。为了提高效率，我们必须积极采取措施减少浪费的时间。

什么是救赎时间？“救赎”这个词在我们的字典里并不常见，所以至少让我们看看老韦伯斯特是怎么说的。它列出了许多定义，但其中最突出的两个是，“获得或赢回”和“使有价值”。这两个定义是我们的短语“赎回时间”的完美结合。前者假设某些东西被拿走了，并号召我们采取行动把它赢回来。第二种观点认为，虽然必要，但不值得，除非我们让它变得必要。我又飘到平流层了，所以让我们用降落伞降落，把着陆点固定在好的旧水泥地上。

假设你开车去上班，去健身房，去姻亲家，或者其他任何你在车里的地方。这次你计划好了吗？还是坐在那里等你赢回来，让它变得值得？当然，目的地可能是值得的，但是花在车上的时间，就其本身而言，不考虑目的地，值得吗？还是需要把韩元要回来？要做到这一点，我们可以有一个播客播放列表；我们计划好的事情，我们有意提前决定的值得我们花费时间的事情，而不是 feed 中任何令人上瘾的建议。你必须事先决定什么是值得你花时间的，这样你才能把它赎回来。
