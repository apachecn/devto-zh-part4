# iDevice Pro 提示:通过一次滑动选择多封邮件💬

> 原文：<https://dev.to/king/idevice-pro-tip-selecting-multiple-messages-with-a-single-swipe-37c0>

## 抽象✳️

***这里有一个简单的 iOS/iPadOS Pro 提示，你可以把它添加到你的技能清单中，并开始在日常生活中使用——希望能节省你的时间，让你的生活更方便*** 。

# 介绍🔦

当在 iPhone 或 iPad 上的给定对话中选择一系列信息时，无论是你自己的还是收件人的，你都可以选择使用一种技术来加快整个过程，这就是它。

# 操作方法📲

在启动选择过程后，或者通过用两个手指在 iOS 13 上双击**，或者通过久经考验的方法**长按非特定消息**，您会看到两个选项:**

**以新月形或半圆形(从右到左/从左到右)向上滑动，形成**——比如:

[![Upwards crescent (half-circle) swipe animation](img/bf7f03a20bb3e6bb606956a7a7e70491.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--7V5LXtAO--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/qh2am8l7by180snhabnc.gif)

…或者，**直接向左/向右滑动，然后向上滑动**—例如:

[![Upwards ninety-degree turn swipe animation](img/42f2fbd02236159b5c367cfe4cc45884.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--4wbB9Jps--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/9m8adp6m2xth6u3ftog9.gif)

在那里，您可以**删除所选的消息**，或者继续执行弹出菜单中显示的其他任意操作。
