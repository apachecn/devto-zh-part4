## 自定义救援类型。

让我们去掉布尔的限制，适当地设计状态。事实是我们用户可以有不同的类型。所以合适的型号应该是

```
type User = {
  id: string
  name: string
  userType: UserType 
}
type UserType = 'Admin' | 'Normal' | 'Moderator' | 'Manager' 
/* Yes, UserType can be also represented as Enum type */ 
```
